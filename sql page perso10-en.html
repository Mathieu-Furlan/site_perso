Tenth case: Select only the top 5 results for each id.

What we have:

donnee
annee id taux
2006 p01 8
2003 p01 7,4
2008 p01 6,8
2001 p01 5,9
2007 p01 5,3
2009 p01 4,4
2002 p01 3,9

2000 p01 0,8
2001 p02 12,5
2004 p02 12,4
2002 p02 12,2
2003 p02 10,3
2000 p02 8,7
2006 p02 4,6

What we would like to achieve:

result
annee id taux
2006 p01 8
2003 p01 7,4
2008 p01 6,8
2001 p01 5,9
2007 p01 5,3
2001 p02 12,5
2004 p02 12,4
2002 p02 12,2
2003 p02 10,3
2000 p02 8,7

And finally the code to obtain above result

SELECT a.annee, a.id, a.taux
FROM donnee AS a
WHERE a.taux IN(
SELECT TOP 5 b.taux
FROM donnee b
WHERE a.id = b.id
ORDER BY b.taux DESC);
