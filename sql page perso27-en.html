What we would like is to fetch rows where test 1 score is the highest by week and student.

Test 2 value is the score obtain by same class.

We want to add a column displaying the score of class H:

result
week student classe H test1 test2
1 1 A 19 93 97
1 2 A 61 88 56
1 3 B 63 76 97
2 1 B 74 77 80
2 2 B 54 92 98
2 3 C 86 90 72

The query to obtain above result:

SELECT b.week, b.student, b.classe, (SELECT Max(score) FROM donnee WHERE class = "H" AND week = a.week AND student = a.student) AS H, b.test1, c.score AS test2
FROM (SELECT week, student, (SELECT Min(class) FROM donnee WHERE score = test1) AS classe, test1 FROM (SELECT week, student, Max(score) AS test1 FROM donnee AS a WHERE test = 1 GROUP BY week, student) AS [%$##@_Alias]) AS b, (SELECT * FROM donnee WHERE test = 2) AS c
WHERE b.week = c.week AND b.student = c.student AND b.classe = c.class;
