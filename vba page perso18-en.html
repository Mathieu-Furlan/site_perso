Heighteenth case: Find cell's number in which value is the date of today.

Summary of the problem:

I am trying to locate the celle in which is entered the date of today.

Say today date is 22 April 2022, I need the macro to return an integer wich indicates the position of the cell in the row.

Create a set of random data:

	
	Sub populatedates()

	Dim i As Integer

	Range("A1", Columns(Cells(1, Columns.Count).End(xlToLeft).Column)).Font.Color = vbBlack

	For i = 1 To 30

		Cells(1, i) = (Date - 3) + i

	Next i

	End Sub

How to do what is asked:

	
	Sub trouverdate()

	Dim d As Range

	Set d = Range("A1", Columns(Cells(1, Columns.Count).End(xlToLeft).Column)).Find(Date, LookIn:=xlValues)

	If Not d Is Nothing Then
		d.Font.Color = vbRed
	Else
		MsgBox "Date not found"
    
	End If

	End Sub
