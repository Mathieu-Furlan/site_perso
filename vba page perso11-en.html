Eleventh case: adding a comment in a cell based on another cell's value.

Summary of the problem:

I want to add a comment into comment section of a cell based on another cell's value.

If the other cell is empty, "Nothing" should be added as comment.

Code to be put at worksheet level to automate the process:

	
	Private Sub Worksheet_Change(ByVal Target As Range)

	If Not Cells(Target.Row, 2).Comment Is Nothing Then
		Cells(Target.Row, 2).Comment.Delete
	End If

	If Target <> "" Then
		Cells(Target.Row, 2).AddComment (Target)
    
	ElseIf Target = "" Then
		Cells(Target.Row, 2).AddComment "Nothing"
    
	End If
  
	End Sub

Code to do what was asked:

	
	Sub commentaire()

	Dim a As Variant

	a = Cells(2, 6)

	If Not Cells(2, 2).Comment Is Nothing Then
		Cells(2, 2).Comment.Delete
	End If

	If a <> "" Then
		Cells(2, 2).AddComment (a)
    
	ElseIf a = "" Then
		Cells(2, 2).AddComment "Nothing"
    
	End If
  
	End Sub
