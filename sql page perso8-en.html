Eighth case: Return a value depending on a column's values

What we have:

donnee
CREDIT_ID FACIL_ID SCORE_TYPE
11111 23 G
11111 25 G
11111 26 G
11111 27 G
22222 67 G
22222 68 B
22222 69 B
22222 70 Z

We would like to display for each credit_id the score_type if it is the same for all lines with a defined credit_id.
Else NULL should be returned.

Result
CREDIT_ID rand
11111 G
22222 NULL

Below is the code to obtain such result:

SELECT b.CREDIT_ID, IIf(Scoretype<=1,SCORE_TYPE,"NULL") AS rand
FROM (SELECT a.CREDIT_ID, Count(a.SCORE_TYPE) AS Scoretype FROM (SELECT DISTINCT CREDIT_ID, SCORE_TYPE FROM donnee) AS a GROUP BY a.CREDIT_ID ORDER BY a.CREDIT_ID) AS c INNER JOIN donnee AS b ON c.CREDIT_ID = b.CREDIT_ID
GROUP BY b.CREDIT_ID, IIf(Scoretype<=1,SCORE_TYPE,"NULL");
