Problèmes VBA et leurs solutions

Seizième cas: cacher une ligne si une valeur définie et trouvée dans une colonne.

Résumé du problème:

Je voudrais masquer une ligne si la chaîne de charactères "work1" est trouvée dans une colonne.

C'est une feuille de travail interactive qui peut avoir plusieurs filtres appliqués en même temps.

Macro pour remplir la feuille excel de données aléatoires:

	
	Sub populatecolumn()

	Dim i As Integer

	For i = 1 To 10
    
		Rows(i).Hidden = False
		Cells(i, 6) = "Work" & Int((Rnd * 3) + 1)

	Next i

	End Sub

Macro permettant d'apporter une solution:

	
	Sub trouvermot()

	Dim i As Integer

	For i = 1 To Cells(Rows.Count, 6).End(xlUp).Row

		If InStr(Cells(i, 6), "Work1") <> 0 Then
			Rows(i).Hidden = True
		End If

	Next i

	End Sub
