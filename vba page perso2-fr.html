Second cas: comment remplacer des cellules vides ou contenant 0 dans une colonne en fonction de valeurs dans une autre colonne.

Résumé du problème:

Il y a une liste de nombres et à côté une lettre qui détermine la catégorie de la ligne, Comment modifier les valeurs manquantes ou égales à 0 avec une formule en fonction de la catégorie?

Nous avons une liste de prix et de poids pour les produits a, b et c. Les prix moyens des produits sont déjà connus et sont dans un tableau sur la même feuille excel.

Pour remplir les cellules sans valeur avec une estimation du poids du produit, quel serait le code?

La subroutine (qui rempli aléatoirement les colonnes de façon à pour rélaiser des tests):

	
	Sub letters()

	Dim i As Integer
	Dim val As Integer
	Dim j As Integer

	Randomize

	For i = 1 To 100
		Cells(1 + i, 1) = Application.Round(Rnd * 10000, 0)
		Cells(1 + i, 2) = Application.Round(Rnd * 10000, 0)
     
	Next i

	For q = 1 To 100

		val = Int((3 * Rnd) + 1)
    
		If val = 1 Then
			Cells(1 + q, 3) = "a"
        
		ElseIf val = 2 Then
			Cells(1 + q, 3) = "b"
        
		ElseIf val = 3 Then
			Cells(1 + q, 3) = "c"
        
		End If
    
	Next q
        
	For j = 1 To Range("A2", Range("A2").End(xlDown)).Rows.Count

	Debug.Print (j)

		If Cells(1 + j, 1) < 1000 Then
			If Cells(1 + j, 3) = "a" Then
				Cells(1 + j, 1) = 1200
        
			ElseIf Cells(1 + j, 3) = "b" Then
				Cells(1 + j, 1) = 1500
            
			ElseIf Cells(1 + j, 3) = "c" Then
				Cells(1 + j, 1) = 1800
            
			End If
		End If
    
	Next j


	End Sub
