Problèmes VBA et leurs solutions

Troisième cas: concaténer n'importe quel nombre de ligne en trois colonnes.

Résumé du problème:

Je travail sur une extraction excel. Il s'agit simplement de plusieurs colonnes avec quelques lignes pour chacune.

Je voudrais applanir ces données en trois colonnes.

Le code pour réaliser ce qui est demandé:

	
	Sub flatten()

	Dim i As Integer
	Dim q As Integer
	Dim a As Integer
	Dim b As Integer
	Dim u As Integer

	a = Cells(Rows.Count, "A").End(xlUp).Row
	b = Int(Cells(1, Columns.Count).End(xlToLeft).Column) / 3

	Debug.Print (b)

	For u = 1 To b - 1
		For i = 1 To 3
			For q = 1 To a
        
				Cells(a * u + q, i) = Cells(q, i + (u * 3))
				Cells(q, i + (u * 3)).ClearContents
        
			Next q
		Next i
	Next u

	End Sub
