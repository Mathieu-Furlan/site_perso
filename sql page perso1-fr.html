Premier cas: trier par différences en pourcentage

Situation initiale:

Table1
NumberA NumberB
105 100
120 110
115 134
140 170
140 190

Ce que nous voulons obtenir:

Result
NumberA NumberB Range
105 100 0-6%
120 110 7-15%
115 134 15-20%
140 170 More than 20%
140 190 More than 20%

Pour finir le code utilisé pour obtenir les résultats:

SELECT Table1.*, SWITCH(abs(NumberB/NumberA-1)<=0.06,"0-6%",abs(NumberB/NumberA-1)<=0.15,"7-15%",abs(NumberB/NumberA-1)<=0.2,"15-20%",abs(NumberB/NumberA-1)>0.2,"More than 20%") AS Range
FROM Table1;
