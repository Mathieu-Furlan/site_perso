Second case: how to replace blank or 0 values in column depending on value in another column.

Summary of the problem:

There is a list of numbers and next to it a letter to determine what category it's in, how would I be able to change blank or 0 values with a formula depending on the category?

In this case there's a list of price and weight for product a, b and c. The average prices for the products are already known and are in a table on the same excel sheet.

To fill in the 0 data with an estimate of how much the product would've weighted, what would the code look like?

The code (which populate columns randomly for the purpose of testing) to do what is asked:

	
	Sub letters()

	Dim i As Integer
	Dim val As Integer
	Dim j As Integer

	Randomize

	For i = 1 To 100
		Cells(1 + i, 1) = Application.Round(Rnd * 10000, 0)
		Cells(1 + i, 2) = Application.Round(Rnd * 10000, 0)
     
	Next i

	For q = 1 To 100

		val = Int((3 * Rnd) + 1)
    
		If val = 1 Then
			Cells(1 + q, 3) = "a"
        
		ElseIf val = 2 Then
			Cells(1 + q, 3) = "b"
        
		ElseIf val = 3 Then
			Cells(1 + q, 3) = "c"
        
		End If
    
	Next q
        
	For j = 1 To Range("A2", Range("A2").End(xlDown)).Rows.Count

	Debug.Print (j)

		If Cells(1 + j, 1) < 1000 Then
			If Cells(1 + j, 3) = "a" Then
				Cells(1 + j, 1) = 1200
        
			ElseIf Cells(1 + j, 3) = "b" Then
				Cells(1 + j, 1) = 1500
            
			ElseIf Cells(1 + j, 3) = "c" Then
				Cells(1 + j, 1) = 1800
            
			End If
		End If
    
	Next j


	End Sub
