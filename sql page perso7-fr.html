Septième cas: Obtnir les résultats d'une table en fonction des valeurs dans une autre

Situation de départ:

project
Id Name
1 ABC
2 DEF
3 GHI
4 JKL
5 MNO
6 PQR

attachment
Id Status ProjectId
1 a1 1
2 a1 1
3 a2 2
4 a2 2
5 a1 3
6 a2 3
7 a1 4
8 a2 4
9 a3 4
10 a1 5
11 a2 5
12 a4 5

Nous voudrions afficher les noms de projet qui ont un statut attaché a1 et a2 (ils doivent avoir les deux) et aucun statut attaché a3 ou a4.

Le résultat devrait être: GHI

Ci-dessous le code pour arriver au résultat:

SELECT a.Name
FROM project AS a
WHERE EXISTS (SELECT Status FROM attachment b WHERE b.ProjectId = a.Id AND status = "a1") AND EXISTS (SELECT Status FROM attachment b WHERE b.ProjectId = a.Id AND status = "a2")
AND NOT EXISTS(SELECT Status FROM attachment b WHERE b.ProjectId = a.Id AND status = "a3") AND NOT EXISTS(SELECT Status FROM attachment b WHERE b.ProjectId = a.Id AND status = "a4")
GROUP BY a.Name;
