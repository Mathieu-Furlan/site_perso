Problèmes VBA et leurs solutions

Quatorzième cas: décaler un interval de données.

Résumé du problème:

Je voudrais selectionner un interval nommé, en faire la copie qui serait ensuite collée ailleurs sur la feuille de travail.

Comment réaliser ce qui est demandé:

	
	Sub deplacerplage()

	Dim maplage As Range
	Dim deux As Range
	Dim i As Integer

	Set maplage = Range("A1:D1")
	Set deux = Range("F1:K1")

	For i = 1 To 10

	maplage.Offset(i, 0) = maplage.Value
	deux.Offset(i, 0) = deux.Value

	Next i

	End Sub
