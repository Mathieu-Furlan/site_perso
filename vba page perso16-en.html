Sixteenth case: hide row if value is found in column.

Summary of the problem:

I want to hide rows if the particular string "work1" is found in a column.

It is an interactive table so it can have different filters applied at a time.

Code to populate the worksheet:

	
	Sub populatecolumn()

	Dim i As Integer

	For i = 1 To 10
    
		Rows(i).Hidden = False
		Cells(i, 6) = "Work" & Int((Rnd * 3) + 1)

	Next i

	End Sub

Code to solve the case

	
	Sub trouvermot()

	Dim i As Integer

	For i = 1 To Cells(Rows.Count, 6).End(xlUp).Row

		If InStr(Cells(i, 6), "Work1") <> 0 Then
			Rows(i).Hidden = True
		End If

	Next i

	End Sub
