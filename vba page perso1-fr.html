Problèmes VBA et leurs solutions

Premier cas: supprimer une ligne si une valeur est dans une liste, parcourir la liste avec une boucle.

Résumé du problème:

J'ai deux tableaux. Une tableau s'appelle DRData, l'autre CheckData. EANCODE est la colonne J pour DRData, et la colonne A pour Checkdata.

Je veux vérifier si CheckData.EANCODE est présent dans DRData.EANCODE. Si c'est le cas; la ligne doit être supprimée de CheckData.

Quand EANCODE est absent, je voudrais laisser une ligne vide pour vérifier ce code. Au final une liste dans CheckData doit contenir les valeurs EANCODE qui ne sont pas dans DRData.

Le code (qui génère des nombres aléatoires pour effectuer des tests) pour faire ce qui est demandé:

	
	Sub ean()

	Dim j As Integer
	Dim l As Integer
	Dim i As Integer
	Dim q As Integer
	Dim x As Long
	Dim u As Integer

	Randomize

	For j = 1 To 100

		Worksheets("DRData").Cells(j, 2) = Application.Round(Rnd * 100, 0)
    
	Next j

	For l = 1 To 100

		Worksheets("CheckData").Cells(l, 5) = Application.Round(Rnd * 100, 0)
    
	Next l


	For i = 1 To 100

		x = Worksheets("CheckData").Cells(i, 5)
    
		For q = 1 To 100
    
			If Worksheets("DRData").Cells(q, 2) = x Then Worksheets("CheckData").Cells(i, 5) = ""
        
		Next q
    
	Next i

	For u = i To 1 Step -1

		If IsEmpty(Worksheets("CheckData").Cells(u, 5)) Then Worksheets("CheckData").Rows(u).Delete
    
	Next u
    
	End Sub
