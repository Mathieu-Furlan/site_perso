Ce que nous voulons atteindre:

result
id a score b score c score
1 88 97 92
2 100 99 85
3 95 87 96
4 90 90 86
5 95 78 98

Première solution utilisant des sous-rquêtes:

SELECT id, (SELECT Max(score) FROM donnee WHERE id = a.id AND version = "a") AS [a score], (SELECT Max(score) FROM donnee WHERE id = a.id AND version = "b") AS [b score], (SELECT Max(score) FROM donnee WHERE id = a.id AND version = "c") AS [c score]
FROM donnee AS a
GROUP BY id;

Seconde solution utilisant pivot:

TRANSFORM Max(score)
SELECT id
FROM donnee
GROUP BY id
PIVOT version;
