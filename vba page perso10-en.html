VBA problems and their solutions

Tenth case: create a tally column.

Summary of the problem:

I would like to count how many times a value is in the column "furniture", however this column may sometimes be in a different place.

So I'd have to refer to the name instead of column number.

Eventually create a new column where it is stated how many times that furniture appears.

Code to randomly generate data:

	
	Sub PopulateFurniture()

	Dim i As Integer
	Dim a As Byte

	For i = 2 To 30
		a = Int((Rnd * 6) + 1)
			If a = 1 Then
				Cells(i, 2) = "chair"
			ElseIf a = 2 Then
				Cells(i, 2) = "table"
			ElseIf a = 3 Then
				Cells(i, 2) = "lamp"
			ElseIf a = 4 Then
				Cells(i, 2) = "sofa"
			ElseIf a = 5 Then
				Cells(i, 2) = "trunk"
			ElseIf a = 6 Then
				Cells(i, 2) = ""
			End If
	Next i

	End Sub

How to solve the problem:

	
	Sub comptermeuble()

	Dim d
	Dim i As Integer
	Dim q As Integer

	Set d = CreateObject("Scripting.Dictionary")

	For i = 2 To Cells(Rows.Count, 2).End(xlUp).Row
		If Not d.Exists(Cells(i, 2).Value) And Not IsEmpty(Cells(i, 2)) Then
			d.Add (Cells(i, 2).Value), Application.CountIf(Columns(2), Cells(i, 2))
    
		End If
	Next i

	For q = 2 To Cells(Rows.Count, 2).End(xlUp).Row

		Cells(q, 3) = d(Cells(q, 2).Value)

	Next q

	End Sub
