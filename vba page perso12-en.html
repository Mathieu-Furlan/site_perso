VBA problems and their solutions

Twelfth case: selecting a fixed number of values in column.

Summary of the problem:

I have a table with a large amount of information, how do I select just the last month worth?

(i.e. just the last 31st cells in the column?)

Code to randomly create date:

	
	Sub CreerDates()

	Dim i As Integer
	Dim mois
	Dim jour

	Randomize
	For i = 1 To 40

	mois = Int((Rnd * 12) + 1)
	jour = Int((Rnd * 31) + 1)

	If mois < 10 Then
		mois = 0 & mois
	End If

	If jour < 10 Then
		jour = 0 & jour
	End If

	Cells(i, 1) = jour & "/" & mois & "/" & Int((Rnd * 2022) + 1)
	Cells(i, 2) = Int((Rnd * 40) + 1)

	Next i

	End Sub

How to select last 31st entries in column:

	
	Sub Derniermois()

	Dim y As Integer
	Dim x As Integer

	x = Cells(Rows.Count, 1).End(xlUp).Row

	y = x - 31

	Range(Cells(y, 1), Cells(x, 2)).Select

	End Sub
