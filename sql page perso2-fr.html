Deuxième cas: différents types de sommes

situation initiale:

donnee
name age sal month
A 21 100 JAN
B 22 150 FEB
C 21 200 SEP
D 22 100 OCT
EMP 21 150 DEC

Ce que l'on souhaite obtenir:

Normal Sum
name age sal tsum
A 21 100 700
B 22 150 700
C 21 200 700
D 22 100 700
EMP 21 150 700

Le code pour obtenir le résultat ci-dessus:

SELECT a.name, a.age, a.sal, b.tsum
FROM donnee AS a, (SELECT Sum(sal) AS tsum FROM donnee) AS b;

Running Sum
name age sal total
A 21 100 100
B 22 150 250
C 21 200 450
D 22 100 550
EMP 21 150 700

Le code pour obtenir le résultat ci-dessus:

SELECT a.name, a.age, a.sal, (SELECT SUM(b.sal) FROM donnee b WHERE b.name <= a.name) AS total
FROM donnee AS a;
