First case: delete row if value is in list, looping through list.

Summary of the problem:

I have two tables. One table is called DRData, other table is CheckData. EANCODE is Column J for DRData, and Column A for Checkdata.

I want to check whether CheckData.EANCODE is present in DRData.EANCODE. If so; delete that row from CheckData.

When the EANCODE is not present, I want to hop over a row to check that code. I want to end with a list in CheckData where all the EANCODE values that are not in DRData are shown.

The code (which generate random numbers for the purpose of testing) to do what is asked:

	
	Sub ean()

	Dim j As Integer
	Dim l As Integer
	Dim i As Integer
	Dim q As Integer
	Dim x As Long
	Dim u As Integer

	Randomize

	For j = 1 To 100

		Worksheets("DRData").Cells(j, 2) = Application.Round(Rnd * 100, 0)
    
	Next j

	For l = 1 To 100

		Worksheets("CheckData").Cells(l, 5) = Application.Round(Rnd * 100, 0)
    
	Next l


	For i = 1 To 100

		x = Worksheets("CheckData").Cells(i, 5)
    
		For q = 1 To 100
    
			If Worksheets("DRData").Cells(q, 2) = x Then Worksheets("CheckData").Cells(i, 5) = ""
        
		Next q
    
	Next i

	For u = i To 1 Step -1

		If IsEmpty(Worksheets("CheckData").Cells(u, 5)) Then Worksheets("CheckData").Rows(u).Delete
    
	Next u
    
	End Sub
