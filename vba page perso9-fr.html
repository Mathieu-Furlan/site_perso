Neuvième cas: selectionner un interval négatif.

Résumé du problème:

Premièrement il faudrait trier la colonne "B". Ensuite il faut selectionner l'interval de valeurs négatives.

Note: les données seront dynamiquement obtenues, le nombre de ligne n'est pas fixe.

Code pour générer des données aléatoires:

	
	Sub PopulateColumns()

	Dim i As Integer
	Dim q As Integer

	Randomize

	For i = 1 To 3
		For q = 2 To 30
			Cells(q, i) = Int((((-1 * Rnd) + Rnd) * 30) + 1)
        
		Next q
	Next i

	End Sub

Code pour selectionner les valeurs négatives:

	
	Sub trier()

	Dim i As Integer

	Range("B2", Range("B2").End(xlDown)).Sort Key1:=Columns(2), Order1:=xlAscending, Header:=xlNo

	i = 2

	Do While Cells(i, 2).Value < 0
		i = i + 1
    
	Loop

	If i > 2 Then
		Range("B2", "B" & i - 1).Select
    
	End If

	End Sub
