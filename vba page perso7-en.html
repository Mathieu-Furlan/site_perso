Seventh case: hide columns based on condition.

Summary of the problem:

What I would like to do is hide two columns if there is a defined value in another.

In our exemple (which generate random values for testing) if a cell in comlumn "A" contains the value "3", columns "B" and "C" will be hidden:

Macro to randomly populate columns:

	
	Sub populatecolumns()

	Dim i As Integer

	Randomize

	For i = 1 To 30

	Cells(i, 1) = Int((Rnd * 30) + 1)
	Cells(i, 2) = Int((Rnd * 30) + 1)
	Cells(i, 3) = Int((Rnd * 30) + 1)

	Next i

	End Sub

Macro to hide columns:

	

	Sub cacher()

	Dim q As Integer

	Columns("B:C").Hidden = False

	For q = 1 To Cells((Rows.Count), 1).End(xlUp).Row

    If Cells(q, 1) = 3 Then
        Columns("B:C").Hidden = True
        
    End If
    
	Next q

	End Sub

Modifying values in column "A" triggers the macro(code to be added directly to the worksheet):

	
	Private Sub Worksheet_Change(ByVal Target As Range)

	If Not Intersect(Target, Range("A1:A30")) Is Nothing Then
		Call cacher
    
	End If

	End Sub
