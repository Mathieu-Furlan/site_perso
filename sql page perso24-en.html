Twenty-fourth case: inventory of status based on one column.

What we have:

donnee
App No Dept Status
123 IT Pending
124 Sales Closed
456 Sales Declined
821 IT Pending
221 IT Closed
876 Sales Pending
665 Sales Closed

What we would like:

result
dept closed pending declined
IT 1 2 0
Sales 2 1 1

And the code to achieve above result:

SELECT dept, (SELECT Count(status) FROM donnee WHERE dept = a.dept AND status = "closed") AS closed, (SELECT Count(status) FROM donnee WHERE dept = a.dept AND status = "pending") AS pending, (SELECT Count(status) FROM donnee WHERE dept = a.dept AND status = "declined") AS declined
FROM donnee AS a
GROUP BY dept;
