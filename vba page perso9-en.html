Ninth case: select negative range.

Summary of the problem:

First column "B" has to be sorted. Then I need to select the range of negative values.

Note: Obtained data will be dynamic, number of rows is not fixed.

Code to populate the worksheet randomly:

	
	Sub PopulateColumns()

	Dim i As Integer
	Dim q As Integer

	Randomize

	For i = 1 To 3
		For q = 2 To 30
			Cells(q, i) = Int((((-1 * Rnd) + Rnd) * 30) + 1)
        
		Next q
	Next i

	End Sub

Code to select negative values:

	
	Sub trier()

	Dim i As Integer

	Range("B2", Range("B2").End(xlDown)).Sort Key1:=Columns(2), Order1:=xlAscending, Header:=xlNo

	i = 2

	Do While Cells(i, 2).Value < 0
		i = i + 1
    
	Loop

	If i > 2 Then
		Range("B2", "B" & i - 1).Select
    
	End If

	End Sub
