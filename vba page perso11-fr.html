Problèmes VBA et leurs solutions

Onzième cas: ajouter un commentaire dans une cellule en fonction de la valeur d'une autre cellule.

Résumé du problème:

Je voudrais ajouter un commentaire à une cellule en fonction de la valeur contenue dans une autre cellule.

Si l'autre cellule est vide, "Nothing" sera ajouté en tant que commentaire.

COde à intégrer au niveau de la feuille excel pour automatiser la tâche:

	
	Private Sub Worksheet_Change(ByVal Target As Range)

	If Not Cells(Target.Row, 2).Comment Is Nothing Then
		Cells(Target.Row, 2).Comment.Delete
	End If

	If Target <> "" Then
		Cells(Target.Row, 2).AddComment (Target)
    
	ElseIf Target = "" Then
		Cells(Target.Row, 2).AddComment "Nothing"
    
	End If
  
	End Sub

Code pour répondre à la demande:

	
	Sub commentaire()

	Dim a As Variant

	a = Cells(2, 6)

	If Not Cells(2, 2).Comment Is Nothing Then
		Cells(2, 2).Comment.Delete
	End If

	If a <> "" Then
		Cells(2, 2).AddComment (a)
    
	ElseIf a = "" Then
		Cells(2, 2).AddComment "Nothing"
    
	End If
  
	End Sub
