Seventeenth case: find highest value in a column.

Summary of the problem:

I am trying to find out which row has the highest number in column "A"

First the code to create random data:

	
	Sub populatecolumn()

	Dim i As Integer

	Randomize

	For i = 1 To 30

		Cells(i, 1) = Int((Rnd * 500) + 1)
    
	Next i

	End Sub

Below is our solution:

	
	Sub trouverplusgrand()

	Dim x As Double
	Dim i As Integer

	x = Application.Max(Range("A:A"))

	For i = 0 To ActiveSheet.UsedRange.Rows.Count

		If Range("A1").Offset(i, 0) = x Then
			MsgBox "Maximum value is in row " & i + 1, vbInformation
			Exit For
		End If
	Next

	End Sub
