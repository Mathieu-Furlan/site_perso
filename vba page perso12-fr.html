Problèmes VBA et leurs solutions

Douzième cas: selectionner un nombre défini de valeurs dans une colonne.

Résumé du problème:

J'ai un tableau qui contient un grand nombre d'informations, comment selectionner uniquement ce qui correspond au dernier mois?

(c.a.d. uniquement les dernières 31 cellules de la colonne?)

Code pour créer aléatoirement des dates:

	
	Sub CreerDates()

	Dim i As Integer
	Dim mois
	Dim jour

	Randomize
	For i = 1 To 40

	mois = Int((Rnd * 12) + 1)
	jour = Int((Rnd * 31) + 1)

	If mois < 10 Then
		mois = 0 & mois
	End If

	If jour < 10 Then
		jour = 0 & jour
	End If

	Cells(i, 1) = jour & "/" & mois & "/" & Int((Rnd * 2022) + 1)
	Cells(i, 2) = Int((Rnd * 40) + 1)

	Next i

	End Sub

Comment selectionner les dernières 31 valeurs d'une colonne:

	
	Sub Derniermois()

	Dim y As Integer
	Dim x As Integer

	x = Cells(Rows.Count, 1).End(xlUp).Row

	y = x - 31

	Range(Cells(y, 1), Cells(x, 2)).Select

	End Sub
