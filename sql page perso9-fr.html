Neuvième cas: Selectionner seulement la valeur la plus récente pour chaque identfiant.

Situation de départ:

donnee
ean price dtmode
155105 2,19 22/06/2022 03:03:25
155105 -0,01 28/06/2022 02:27:15
155105 1,45 28/06/2022 15:11:35
114695 -0,01 28/06/2022 02:27:15
114695 5,99 28/06/2022 15:11:27
213786 -0,01 28/06/2022 02:27:15
213786 2,39 28/06/2022 15:11:32

Ce que nous voulons:

result
ean price dtmode
114695 5,99 28/06/2022 15:11:27
155105 1,45 28/06/2022 15:11:35
213786 2,39 28/06/2022 15:11:32

Ci-dessous le code pour obtenir le résultat:

SELECT ean, price, dtmod
FROM donnee AS a
WHERE dtmod = (SELECT Max(dtmod) FROM donnee WHERE ean = a.ean GROUP BY ean)
ORDER BY ean;
