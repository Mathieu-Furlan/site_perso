Fourteenth case: offseting a range of data.

Summary of the problem:

I would like to select a named range, copy it and paste it for a certain number of rows.

How to do what is asked:

	
	Sub deplacerplage()

	Dim maplage As Range
	Dim deux As Range
	Dim i As Integer

	Set maplage = Range("A1:D1")
	Set deux = Range("F1:K1")

	For i = 1 To 10

	maplage.Offset(i, 0) = maplage.Value
	deux.Offset(i, 0) = deux.Value

	Next i

	End Sub
