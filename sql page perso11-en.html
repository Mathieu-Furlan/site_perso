Eleventh case: Remove characters from a string.

What we have:

donnee
ID
60079804-S/O PALMER DONAVIN MFG
60008517-S/O READY CABLE INC.
60008049-S/O ACORN INTERNATIONAL

What we would like to achieve:

result
modified
60079804-PALMER DONAVIN MFG
60008517-READY CABLE INC.
60008049-ACORN INTERNATIONAL

And finally the code to obtain above result

SELECT Replace(Replace(ID,"S/O",""), "- ", "-") AS modified
FROM donnee;
