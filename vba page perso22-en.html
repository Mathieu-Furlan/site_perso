Twenty-second case: Appliying formula to each cells in column.

Summary of the problem:

I have an excel file with a column full of numbers (occasionally there will be some text or a blank cell).

I want to apply a function to that cell that does the following;

If the cell value is blank or text do nothing.

If the value is less than 8 then make it 8, if it is greater than or equal to 8 do nothing.

I wish to overwrite the values in the cell with the new values as opposed to preservering them.

First we create the function:

	
	Function plusquehuit(plage As Range) As Variant

    If IsNumeric(plage) And plage <> "" Then
        If plage < 8 Then
            plusquehuit = 8
        Else
            plusquehuit = plage
        End If
    ElseIf Not IsNumeric(plage) Then
        plusquehuit = plage
    Else
        plusquehuit = ""
    End If
	End Function

Then the subroutine:

	
	Sub remplacerhuit()

	Dim i As Integer

	For i = 1 To Cells(Rows.Count, 1).End(xlUp).Row

		Cells(i, 2).Value = plusquehuit(Cells(i, 1))
		Cells(i, 1) = Cells(i, 2)
		Cells(i, 2).ClearContents

	Next i

	End Sub
