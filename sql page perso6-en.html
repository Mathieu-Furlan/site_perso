Sixth case: combine two columns based on data inside them

What we have:

donnee
Article Desc UM Value
item1 Flower PC 5001
item2 Paper PC 6001
item2 Paper CT 6002
item3 Pen CT 7001
item4 Eraser PC 8001

What we would like to achieve:

Result
Article Desc Value PC ValueCT
item1 Flower 5001 NULL
item2 Paper 6001 NULL
item2 Paper NULL 6002
item3 Pen NULL 7001
item4 Eraser 8001 NULL

And finally the code to obtain above result

SELECT Article, Desc, IIf(UM="PC",Value,"NULL") AS [Value PC], IIf(UM="CT",Value,"NULL") AS ValueCT
FROM donnee;
