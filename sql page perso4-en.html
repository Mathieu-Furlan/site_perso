Fourth case: sum values when names are equal

What we have:

donnee
nom valeur
German 4
French 1
FrenchSecond 1
English 2

What we would like to achieve:

Result
nom valeur
English 2
French 2
German 4

And finally the code to obtain above result

SELECT IIf(nom Like "*Second","French",nom) AS nomdef, Sum(valeur) AS somme
FROM donnee
GROUP BY IIf(nom Like "*Second","French",nom);
