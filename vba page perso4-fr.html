Problèmes VBA et leurs solutions

Troisième cas: remplir une colonne en fonction d'une valeur dans une ligne.

Résumé du problème:

J'essaye d'obtenir une macro VBA qui, pour chaque ligne dans un interval donné, si le charactère "Y" apparait ne serait-ce une seul fois, alors la colonne "V" de la même ligne sera remplie de "Y".

Si aucun "Y" est trouvé; c.a.d. si chaque valeur de la ligne est "N", alors la colonne "V" sera remplie de "N".

Le code (qui génère des valeurs aléatoires pour réaliser des tests) qui permet d'obtenir le résultat:

	
	Sub verif()

	Dim i As Integer
	Dim q As Integer
	Dim r As Double
	Dim lastrow As Integer
	Dim j As Integer
	Dim res

	Randomize

	For i = 1 To 15
		For q = 1 To 15
    
		r = Rnd
    
			If r > 0.95 Then
				Cells(i, q) = "Y"
    
			Else
				Cells(i, q) = "N"
            
			End If
    
		Next q
	Next i

	lastrow = Cells(Rows.Count, "A").End(xlUp).Row

	For j = 1 To lastrow
    
		For Each res In Range(Cells(j, 1), Cells(j, 15))
			If res = "Y" Then
				Cells(j, 17) = "Y"
				Exit For
           
			Else: Cells(j, 17) = "N"
       
			End If
       
		Next
    
	Next j
   
	End Sub
