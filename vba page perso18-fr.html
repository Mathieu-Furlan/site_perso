Dix-huitième cas: trouver la cellule qui contient la date d'aujourd'hui.

Résumé du problème:

Je voudrais localiser la cellule qui contient la date d'aujourd'hui.

Mettons que la date d'aujourd'hui soit le 22 avril 2022, je voudrais une macro qui retourne un entier indiquent la position de la cellule dans la ligne.

Créer des dates aléatoires:

	
	Sub populatedates()

	Dim i As Integer

	Range("A1", Columns(Cells(1, Columns.Count).End(xlToLeft).Column)).Font.Color = vbBlack

	For i = 1 To 30

		Cells(1, i) = (Date - 3) + i

	Next i

	End Sub

Obtenir ce qui est demandé:

	
	Sub trouverdate()

	Dim d As Range

	Set d = Range("A1", Columns(Cells(1, Columns.Count).End(xlToLeft).Column)).Find(Date, LookIn:=xlValues)

	If Not d Is Nothing Then
		d.Font.Color = vbRed
	Else
		MsgBox "Date not found"
    
	End If

	End Sub
