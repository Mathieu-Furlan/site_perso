Quatrième cas: additionner les valeurs quand les noms sont identiques

Situation initiale:

donnee
nom valeur
German 4
French 1
FrenchSecond 1
English 2

Ce que nous souhaiterions obtenir:

Result
nom valeur
English 2
French 2
German 4

Le code pour obtnir le résultat ci-dessus:

SELECT IIf(nom Like "*Second","French",nom) AS nomdef, Sum(valeur) AS somme
FROM donnee
GROUP BY IIf(nom Like "*Second","French",nom);
