Third case: concatenate any number of rows into three columns.

Summary of the problem:

I'm working with an extract file in Excel. It's basically multiple columns with several row data on each.

I would like to flatten it into 3 columns.

The code to achieve what is asked:

	
	Sub flatten()

	Dim i As Integer
	Dim q As Integer
	Dim a As Integer
	Dim b As Integer
	Dim u As Integer

	a = Cells(Rows.Count, "A").End(xlUp).Row
	b = Int(Cells(1, Columns.Count).End(xlToLeft).Column) / 3

	Debug.Print (b)

	For u = 1 To b - 1
		For i = 1 To 3
			For q = 1 To a
        
				Cells(a * u + q, i) = Cells(q, i + (u * 3))
				Cells(q, i + (u * 3)).ClearContents
        
			Next q
		Next i
	Next u

	End Sub
