Problèmes VBA et leurs solutions

Dix-septième cas: trouver la valeur la plus élevée d'une colonne.

Résumé du problème:

Je voudrais savoir quelle ligne à la plus haute valeur dans la colonne "A"

Nous générons d'abord des données aléatoires:

	
	Sub populatecolumn()

	Dim i As Integer

	Randomize

	For i = 1 To 30

		Cells(i, 1) = Int((Rnd * 500) + 1)
    
	Next i

	End Sub

Ensuite la solution:

	
	Sub trouverplusgrand()

	Dim x As Double
	Dim i As Integer

	x = Application.Max(Range("A:A"))

	For i = 0 To ActiveSheet.UsedRange.Rows.Count

		If Range("A1").Offset(i, 0) = x Then
			MsgBox "Maximum value is in row " & i + 1, vbInformation
			Exit For
		End If
	Next

	End Sub
