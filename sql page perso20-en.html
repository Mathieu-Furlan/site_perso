Twentieth case: Populate a column based on values in another one.

What we have:

donnee
suite
AK-11 HK-82,7
AK-11 HK-82,8
AK-11 HK-82,9
AK-11 HK-82,10
AK-11 HK-82,11

We would like to isolate the number at the end of each string.

result expr1000 82,7 82,8 82,9 82,10 82,11

Keeping the decimal separator of the system:

SELECT ROUND((CDbl(REPLACE(REPLACE(suite,"AK-","")," HK-",""))-int(CDbl(REPLACE(REPLACE(suite,"AK-","")," HK-",""))/100)*100),2)
FROM donnee;

Forcing the use of dot to separate decimals from integer:

SELECT LTrim(RIGHT(REPLACE(REPLACE(REPLACE(suite,"AK-",""),"HK-",""),",","."),5))
FROM donnee;
