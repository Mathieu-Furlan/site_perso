Fifteenth case: swapping cells until condition is met.

Summary of the problem:

I would like to swap cells through a conditional loop.

What is to be done is to move numeric cells to column "B" and remove non-numeric values.

The solution:

	
	Sub decaler()

	Dim i As Integer
	Dim q As Integer
	Dim j As Integer

	For i = 1 To Cells(Rows.Count, 2).End(xlUp).Row

	q = 1

    If Not IsNumeric(Cells(i, 2)) Then
        Do Until IsNumeric(Cells(i, 2))
        Cells(i, 2) = Cells(i, 2 + q)
        q = q + 1
        Loop
    End If

	Next i

	For j = 3 To Range("A1").SpecialCells(xlCellTypeLastCell).Column
		Columns(j).ClearContents
	Next j

	End Sub
